A WordPress plugin that User role based customizable admin panel.

== Description ==

The " Custom Admin Panel-User Role Based Admin " is a minimal setup custom plugin. The " Custom Admin Panel-User Role Based Admin " features give the ability to customize the WordPress admin login page and few admin areas according to the plugin's settings.

It helps to Customize login page styles and set some custom messages for the WordPress admin panel.
